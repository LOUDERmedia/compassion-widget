The Compassion Widget plug-in displays a banner from Compassion International in a WordPress widget area on your website/blog.

== Description ==

The Compassion Widget plug-in uses the WordPress widget system to insert and display a Compassion International banner.  A user can select from various available Compassion International banners. When a site visitor clicks on a banner they will be taken to the corresponding page on the Compassion International website (http://www.compassion.com).

== Installation ==

1. Upload directory `/compassion-widget` to the `/wp-content/plugins/` directory
2. Activate the plug-in through the 'Plug-ins' menu in WordPress
3. Navigate to Appearance -> Widgets menu in WordPress
4. Drag the 'Compassion Banners' widget from the available widgets box into an available widget area
5. If you have a source code from Compassion, select 'Use Source Code' and enter your source code in the field that appears
6. Optionally select a specific country from which to show children
7. Click 'View Available Banners' to select the banner to be displayed
8. Optionally select a text link to display with the banner
9. Click the save button to save the widget options

== Frequently Asked Questions ==

= What sizes are available for the Compassion Banners? =

Sizes range from 120 px X 60 px up to 425 px X 600 px.

= What if I don't have a widget area big enough to display the banner I have selected? =

Currently these are the only options available to insert the banner into a widget area on your webpage.
